evaluation-trouve-ton-artisan-ThibautDONA

Création du site web "Trouve ton artisan !" permettant de répertorier les artisans de la région Auvergne-Rhône-Alpes (France)

Français

Prérequis

Ce projet utilise les ressources suivantes :

  • git
  • Node.js v24.12.0
  • npm
  • Express
  • Sequelize
  • XAMPP
  • MySQL
  • PhpMyAdmin

Installation

  1. Installez nvm : https://www.nvmnode.com/guide/installation.html

  2. Installez Node.js + npm : Dans le terminal, utilisez :

nvm install 24.12.0

puis :

nvm use 24.12.0

Pour voir si c'est bien installé :

node -v

(Vous devriez voir : v.24.12.0)

  1. Installez git : https://git-scm.com

  2. Clonez le repository : Dans votre terminal, utilisez :

git clone https://github.com/donathibaut/evaluation-api-ThibautDONA
  1. Installez les packages (dont Express) : Avec le terminal, allez à la racine du projet. Dans le terminal, utilisez :
npm i

Ensuite, faites de même dans le répertoire client et le répertoire server.

  1. Installez XAMPP : https://www.apachefriends.org/fr/index.html

  2. Allez sur PhpMyAdmin : Sur l'interface XAMPP, appuyez sur les boutons "Start" des lignes Apache et MySQL. Ensuite, cliquez sur le bouton "Admin" de la ligne MySQL, celle-ci vous enverra sur PhpMyAdmin.

evaluation-trouve-ton-artisan-ThibautDONA

Creation of the website "Trouve ton artisan !" helping you to find craftsmen of the French region Auvergne-Rhône-Alpes

Requirements

This project uses the following resources :

  • git
  • Node.js v24.12.0
  • npm
  • Express
  • Sequelize
  • XAMPP
  • MySQL
  • PhpMyAdmin

Installation

  1. Install nvm : https://www.nvmnode.com/guide/installation.html

  2. Install Node.js + npm : In the terminal, use :

nvm install 24.12.0

then :

nvm use 24.12.0

To see if it's correctly installed :

node -v

(You should see : v.24.12.0)

  1. Install git : https://git-scm.com

  2. Clone this repository : In your terminal, use :

git clone https://github.com/donathibaut/evaluation-api-ThibautDONA
  1. Install the packages (including Express) : With the terminal, go to the project root folder. In the terminal, use :
npm i

Then do the same in the client and the server directory.

  1. Install XAMPP : https://www.apachefriends.org/fr/index.html

  2. Go to PhpMyAdmin : On XAMPP interface, select "Start" buttons in the Apache and MySQL areas. Then, click on the "Admin" button in the MySQL area, this one will redirect you on PhpMyAdmin.
